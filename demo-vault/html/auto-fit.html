Auto-fit to content

This page is taller than the default 400px embed height. With auto-fit the embed grows to fit exactly — no inner scrollbar, no trailing whitespace.

Section 1

Self-contained HTML whose height is not known in advance (dashboards, reports, generated charts) is a good fit for content-based sizing.

Section 2

Each card adds height. A fixed-height embed would either clip this content behind an inner scrollbar or leave a large empty gap for short content.

Section 3

The embed reacts to content-size changes (images finishing loading, expandable sections) via a ResizeObserver.

Section 4

Resize the note pane and the width follows; the height stays fit to the content it renders.

Section 5

This last card pushes the total content height well past 400px so the auto-fit behavior is easy to see.
